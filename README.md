# Assignment 1 - Random Number Generator
___Course___: _Spring 2026 Object Oriented Analy.&Design (CSCI-3612 - 20964)_

___Author:___ _Zaur Rajabov_ 

---

This Java program generates an ArrayList of random doubles, computes statistics about them (size, mean, standard deviation, min, max), and displays them in a tabular format.

To run it, have JDK 21 installed on your machine and run the following commands from the `src` folder:

```
javac Generator.java
java Generator
```
